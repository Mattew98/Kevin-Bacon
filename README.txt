This is a solution to the '6 Degrees of Separation' problem, also known as the '6 Degrees of Kevin Bacon' problem
This version relies on the data that you input. I made this because I liked researching pictures of singers or
actors that I am interested in and I made this one because I can include myself to this one and all the solutions
online only search through celebrities, which I am not.

Use the data writer script to input a new picture entry.

Run the searcher script once all your connected pictures have been made.
